Codespace에서 Copilot 설정

개발 컨테이너(devcontainer)를 만들어 Codespace에 Copilot을 자동 설치합니다.

핵심 파일

.devcontainer/devcontainer.json
Codespace 생성 시 자동으로 Copilot 확장 설치

설정 내용

  • "name" — 구성 이름 지정
  • "customizations.vscode.extensions" — "GitHub.copilot" 추가
  • main 브랜치에 커밋 → Codespace 생성

확인 포인트

Code 탭 → Codespaces → Create codespace → 확장 프로그램 확인

AI 코드 제안 수락하기

함수 헤더를 입력하면 Copilot이 본문을 자동 제안

JavaScript 파일을 만들고 함수 헤더를 입력하면 Copilot이 회색 텍스트로 전체 함수 본문을 제안합니다.

실습 과정

  • skills.js 파일 생성
  • function calculateNumbers(var1, var2) 입력
  • 회색 텍스트로 자동 제안 표시
  • Tab 키로 즉시 수락
Copilot은 Python, JavaScript, TypeScript, Ruby, Go,
C#, C++에서 특히 잘 작동합니다.

코드 푸시

git add → git commit → git push

Copilot 제안 탭 탐색

여러 대안 중 최적의 솔루션을 선택하세요

Copilot은 하나의 제안만 보여주는 것이 아닙니다. Completions Panel에서 약 10가지 대안을 탐색할 수 있습니다.

사용 방법

  • member.js 파일 생성
  • function skillsMember() 입력
  • 빨간 물결선 → … 클릭
  • Open Completions Panel 선택
  • 약 10가지 제안 중 Accept Solution 클릭
첫 번째 제안이 마음에 들지 않을 때
Completions Panel로 더 나은 대안을 찾으세요!

주석으로 코드 생성하기

자연어 주석이 곧 코드가 됩니다

코드 대신 자연어 주석을 입력하면 Copilot이 의도를 파악하여 전체 코드 블록을 생성합니다.

실습 과정

  • comments.js 파일 생성
  • // Create web server 주석 입력
  • Enter로 새 줄 이동 → Copilot이 코드 블록 제안
  • Completions Panel에서 최적 솔루션 선택
주석을 구체적이고 명확하게 작성할수록
Copilot의 제안 품질이 높아집니다.

단축키

Ctrl + Enter = Completions Panel 열기

Copilot 개발 워크플로우

Codespace에서 시작하여 코드 완성까지

devcontainer → Tab 제안 → 제안 탭 → 주석 생성 → Push

달성하는 것

  • 환경 자동화 — devcontainer로 Copilot 자동 설치
  • 인라인 제안 — Tab 한 번으로 코드 수락
  • 대안 탐색 — Completions Panel로 최적 선택
  • 자연어 코딩 — 주석으로 의도 전달, AI가 구현
Copilot은 AI 페어 프로그래머입니다.
코드, 주석, 컨텍스트를 종합하여 최적의 제안을 제공합니다.

GitHub Actions — 실습의 비밀

뒤에서 자동으로 돌아가는 검증 시스템

이 실습은 GitHub Actions를 사용해 여러분의 진행을 자동으로 확인합니다.

동작 원리

  • 1단계: devcontainer 커밋 → push 이벤트 → Codespace 설정 확인
  • 2단계: skills.js 푸시 → push 이벤트 → JS 파일 존재 확인
  • 3단계: member.js 푸시 → push 이벤트 → 제안 탭 활용 확인
  • 4단계: comments.js 푸시 → push 이벤트 → 주석 생성 확인
